Dotfiles
========

Just some of my dotfiles.

Installation
------------

	[options] bash -c "$(curl -fsSL https://raw.github.com/chrisjaure/dotfiles/master/bin/dotfiles)"

Options
-------

- `BOOTSTRAP=[target]` (experimental) where target one of: crostini
